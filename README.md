## Installation

Clone the repository:

```bash
git clone https://github.com/isaac-kow-data-scientist/isaac-kow-react-portfolio.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

Build for production:

```bash
npm run build
```
